### Обо мне

Я разработчик программного обеспечения, специализирующийся на создании производительных и масштабируемых цифровых продуктов. Моя цель — проектирование надежных и интуитивно понятных решений.

- Текущие направления работы: **Высокопроизводительные Telegram-боты и веб-приложения (Web Apps)**
- В работе использую **ИИ-агенты (Codex, Claude Code, GitHub Copilot)** для проектирования архитектуры, прототипирования и оптимизации разработки.
- Область профессиональных интересов: **Внедрение ИИ-решений, продвинутая облачная архитектура и системный дизайн**
- Основной стек: **TypeScript, Node.js, React**
- Рабочее окружение: **15 лет опыта работы с Linux**, в настоящее время основная система — **Arch Linux**

---

### Избранные проекты

- **Алгоритмический бот для криптотрейдинга**: Разработал успешного и производительного бота для алгоритмической торговли, автоматизирующего стратегии на крупных криптовалютных биржах. Создан с упором на скорость и надежность.
  - **Стек технологий:** Python, PostgreSQL, Redis, Docker
- **Telegram-бот для продажи VPN**: Создал автоматизированный магазин в Telegram, который управляет подписками пользователей и бесшовными платежами за личный VPN-сервис. Обеспечивает мгновенный доступ и управление аккаунтом.
  - **Стек технологий:** TypeScript, Node.js, PostgreSQL
